# 🏠 HomeNest - Premium Real Estate Platform

🔗Live link : https://homenest-fahadshakil.netlify.app/

HomeNest is a modern, full-stack MERN real estate application designed to provide a seamless property searching and listing experience. With a focus on premium UI/UX, HomeNest offers verified listings, advanced filtering, and a fully responsive design for both mobile and desktop users.

---

## ✨ Key Features

- **Premium Responsive Design**: A stunning, modern UI built with Tailwind CSS and DaisyUI, optimized for all screen sizes.
- **Dynamic Property Slider**: An interactive home banner featuring gorgeous blue navigation arrows and smooth transitions.
- **Full-Stack Search & Filtering**: Real-time property searching with backend-powered sorting and category filtering.
- **Safe Authentication**: Secure user login and registration powered by Firebase Authentication.
- **Global Dark Mode**: Seamlessly switch between light and dark themes with a stylish theme toggle.
- **User Dashboard**: Authenticated users can manage their own listings, ratings, and profile details.
- **Verified Listings**: A curated collection of the latest properties with detailed information and high-quality imagery.

---

## 🚀 Technologies Used

### Frontend

- **React 19**: Modern component-based architecture.
- **Vite**: Lightning-fast development and build tool.
- **Tailwind CSS 4**: Utility-first styling with advanced design tokens.
- **DaisyUI 5**: Premium component library for high-end aesthetics.
- **Lucide React**: Beautiful, consistent icon set.
- **Swiper.js**: High-performance touch slider for the hero banner.
- **Framer Motion**: Smooth micro-animations and transitions.

### Backend

- **Node.js & Express.js**: Scalable server-side logic and API handling.
- **MongoDB**: Robust NoSQL database for property and user data.
- **Firebase Admin SDK**: Secure server-side authentication verification.
- **Cors & Dotenv**: Environment management and secure cross-origin communication.
